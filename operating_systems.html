🧠 Main Idea

The Operating System works between the user, application software, other system software, and hardware. It manages resources and provides a usable interface.

Exam wording: An OS is system software that manages computer hardware and software resources and provides services to the user and application software.

Booting and Software Types

How the OS starts and where it fits

⚙️ Hardware, Firmware, and Software

TermMeaningExamples / Notes
HardwarePhysical parts of a computer with a definite shape.Mouse, keyboard, monitor, hard disk, speaker, printer.
FirmwareBooting instructions stored in ROM.Initial instructions used when the computer powers on.
SoftwareInstructions given to the computer to perform activities.System software and application software.

🚀 Booting Sequence

  1. Power on: the CPU activates the BIOS.
  2. POST checks whether the hardware is functioning.
  3. The system reads the MBR in the boot drive using the bootstrap loader.
  4. The Operating System is loaded from the boot drive into RAM.
  5. The OS takes control and displays the user interface.
Booting means loading the Operating System into RAM.

🗂️ Software Classification

System SoftwareOperating systems, utility software, and language translators.
Application SoftwareWord processing, spreadsheets, databases, games, and web browsers.
Language TranslatorsAssembler, compiler, and interpreter translate programs to machine language.
Utility SoftwareHelps manage, protect, and analyze the computer.

Operating Systems and Their Types

Examples and classifications

🌍 Examples of Operating Systems

Microsoft WindowsPopular proprietary OS. Examples include Windows XP, Vista, 7, and 8 in the textbook.
Apple Mac OSProprietary OS developed by Apple and used on Apple computers.
UbuntuFree and open-source OS based on Linux.
AndroidMobile operating system developed by Google and based on open-source software.
Hanthana LinuxA Linux-based operating system that can be downloaded free of charge.

📚 Four Main Types

TypeMeaningExample
Single-user OSProvides service to one person at a time.MS-DOS
Multi-user OSAllows multiple users to use a system simultaneously.Linux, Windows Server
Multi-tasking OSAllows multiple processes or tasks to run at the same time.Windows 7/8, Ubuntu, Mac OS
Real-time OSGives output in real time without observable delays.ATM machines, calculators, scientific devices

Services of an Operating System

Resource management and user interface support

🧭 Two Main Service Areas

1. Managing HardwareThe OS manages CPU processes, memory, devices, files, security, and network resources.
2. Providing a User InterfaceThe OS provides a way for users to give commands using CLI or GUI.
The OS coordinates resources such as CPU, RAM, disk, devices, and applications.

Utility Programs and File Systems

Practical O/L file management

🧰 Utility Programs in an OS

Backup SoftwareCopies files and creates backups of drives.
Disk ScannerChecks errors in the hard drive.
DefragmentationRearranges fragmented data to improve disk efficiency.
File CompressionReduces large files into smaller files.
Task ManagerShows processes, programs, and system status.
Anti-virusIdentifies and removes malicious software.
ClipboardTemporarily stores copied or cut data.
Network UtilitiesAnalyze connectivity and configure network settings.

💽 Partitioning, Formatting, Defragmentation

ProcessMeaning
PartitioningDividing one physical drive into multiple logical drives.
FormattingPreparing a storage device so data can be stored using a file system.
DefragmentationRearranging fragmented data so the disk works more efficiently.
Warning: Re-partitioning or formatting can delete existing data. Always make a backup first.

🖼️ Disk Visuals

Partitioning makes logical drives such as C: and D: in Windows.
Defragmentation rearranges file parts so reading becomes faster on affected disks.

📄 File Names, Extensions, and Properties

A file name normally has two parts: file name and extension. The extension helps the OS identify the file type.

ExtensionFile Type
.exeExecutable
.docxWord document
.xlsxExcel spreadsheet
.pptxPowerPoint presentation
.accdbAccess database
The full stop separates the file name from the extension.
File properties can include file type, size, location, created/modified date and time, and storage space used on disk.
