🧭 Main Idea

A logic gate is a small digital circuit that accepts one or more binary inputs and gives one binary output.

Binary values are normally written as 0 and 1. In examples, 0 can mean OFF/closed/no, while 1 can mean ON/open/yes depending on the system.

Signals and Logic Gates

Sections 4.1 and 4.2

4.1 – Signals in Everyday Decisions

Humans use signals to decide what to do. A green signal may mean “go”, a red signal may mean “stop”, and a car warning signal can tell the driver that a door is not closed.

Computers also make decisions using signals. In digital electronics, these signals are represented using binary values: 0 and 1.

4.2 – What is a Logic Gate?

A logic circuit is a circuit designed to build a logical condition using binary values. A logic gate is the basic building block of a logic circuit.

The Central Processing Unit contains a very large number of logic gates. Each gate receives input values and produces an output value.

3-Input Truth Tables

A gate with n inputs has 2n possible input combinations.

3 inputs → 2³ = 8 rows

Example: Q = A.B.C is 1 only when A, B, and C are all 1.

Designing a Circuit from an Expression

Break the expression into smaller parts and build each part with a gate.

  1. Identify the inputs: A and B
  2. Create A+B using an OR gate
  3. Create A.(A+B) using an AND gate
  4. Create final Q=A.(A+B)+B using an OR gate

Integrated Circuits and Practical Usage

Sections 4.6 and 4.7

4.6 – Integrated Circuits

An Integrated Circuit, or IC, is a small packaged circuit made for a specific function. ICs can contain devices such as transistors, resistors, capacitors, and diodes.

Logic gates can be placed inside ICs. A microprocessor is made from many integrated circuits and logic-gate based circuits.
PartRole
Input pinsReceive signals such as A and B
Logic gates inside the ICProcess input signals
Output pinsSend the final output signal Q

🏠 Home Alarm System

A house alarm can use OR logic. If any window or door sensor becomes 1, the alarm output becomes 1.

Alarm = W1 + W2 + FrontDoor + BackDoor
All sensors 0 → no alarm. Any sensor 1 → alarm on.

💡 Street Light Control

A street lamp can use a manual switch, a timer, and a light sensor. One simple logic rule is:

Lamp = Manual + Timer.Dark

This means the lamp turns on manually, or automatically when the timer is active and the environment is dark.
