Types of Number Systems

Section 1.2 & 1.3

What is a Number System?

A number system is a method used to represent numbers. Each system has a fixed number of different symbols. The total count of those symbols is called the base.

The base tells us which number system a number belongs to. It is written as a small subscript on the right of the number.
Binary → 1012, 1110112 (base 2) Octal → 1018, 5738 (base 8) Decimal → 10110, 4710 (base 10) Hexadecimal→ 10116, 7B16 (base 16)

Place Value in Decimal

In the decimal system, every position in a number has a place value that is a power of 10.

Number Conversions

Sections 1.6.1 & 1.6.2

1.6.1 – Decimal → Binary (Divide by 2)

Repeatedly divide the number by 2. Record the remainder each time. Read remainders bottom to top.

  1. Divide the decimal number by 2
  2. Write down the remainder (0 or 1)
  3. Repeat using the quotient until quotient = 0
  4. Read all remainders from bottom to top — that is the binary number
Example: 13₁₀ → Binary
DivisionQuotientRemainder
13 ÷ 261
6 ÷ 230
3 ÷ 211
1 ÷ 201
Read remainders bottom → top: 1101 ∴ 1310 = 11012

1.6.2 – Binary → Decimal (Expand by Place Value)

Multiply each bit by its place value (power of 2) and add all the results.

Example: 10111₂ → Decimal
2⁴ 2³ 2² 2¹ 2⁰ 1 0 1 1 1 = 1×16 + 0×8 + 1×4 + 1×2 + 1×1 = 16 + 0 + 4 + 2 + 1 = 2310
