Chapter roadmap and key ideas

📌 What is Word Processing?

Word processing means creating documents using word processing software. It is used to type, edit, format, save, print, and improve documents neatly.

Instead of preparing documents manually, word processing software helps us make invitations, letters, reports, posters, forms, certificates, and many other documents faster and more attractively.

🧠 Key Terms

Document Cursor Home Tab Font Alignment Word Art Table Spell Check

✅ Skills by the End

  • Create, open, save, save as, close, preview, and print documents
  • Use Cut, Copy, Paste, Undo, Redo, and Spell Check
  • Format text using font, size, colour, bold, italic, underline, highlight, subscript, and superscript
  • Insert pictures, shapes, text boxes, Word Art, clip art, and tables

📄 Documents You Can Make

Examples include letters, question papers, newspapers, application forms, invitations, greeting cards, magazines, and more.

✅ Good Formatting Rules

  • Use bold for headings or important words.
  • Use font colour carefully so text remains readable.
  • Use highlight only for key words.
  • Use superscript and subscript for special text such as 22 and CO2.
