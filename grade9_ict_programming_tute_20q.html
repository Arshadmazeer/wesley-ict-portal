Selection Control Structures

Sections 3.1 & 3.2

3.1 – Simple Selection (if–else)

A simple selection has one condition and two options. If the condition is true, one path is taken; otherwise the other path is taken.

Example: Tossing a coin → "Head" or "Tails"
Start
Toss the coin
Is it
Head?
Yes →
Show "Head"
↓ No
Show "Tails"
End
if Side_landed = Head then say "Head" for 2 secs else say "Tails" for 2 secs

3.2 – Selection Out of Many Options

When you need to choose from more than two options, you chain multiple if–else blocks together.

Example: A dice can show values 1–6. Each value needs its own branch.

The structure checks conditions one by one. The first true condition executes its action; all others are skipped. If none match, the final else runs.

if Value = 1 then say "1" else if Value = 2 then say "2" else if Value = 3 then say "3" else if Value = 4 then say "4" else if Value = 5 then say "5" else say "6"

Repetition Control Structures

Sections 3.3 – 3.6

What is Repetition?

Repetition (a loop) means an action or group of actions is performed again and again. Both the start and end of the loop depend on a condition.

Real-life example: A water pump runs until the tank is full.

3.4 – Three Scratch Repetition Blocks

BlockWhen to useExample
repeat 10 You know exactly how many times to loop Display numbers 1–10
repeat until ◇ Loop until a condition becomes true Display 1–5 (stop when A>5)
forever Loop never stops on its own Display random numbers 1–100 endlessly

🔢 repeat (fixed)

set A to 1 repeat 10 say A wait 1 secs change A by 1

Shows numbers 1 to 10, one per second.

🔁 repeat until (conditional)

set A to 1 repeat until A > 5 say A wait 1 secs change A by 1

Shows 1–5 then stops automatically.

Flowchart for Repetition (Numbers 1–5)

Begin
Number = 1
Number ≤ 5?
↓ Yes
Show Number
Number = Number + 1
↖ loop back ↑
No →
End
The loop variable (Number) must be initialised before the loop and updated inside it, otherwise the loop never ends!

3.6 – Nested Repetition

A nested loop is a loop placed inside another loop. The inner loop completes all its iterations for every single iteration of the outer loop.

repeat 100 ← outer loop (100 times) repeat 4 ← inner loop (4 times per outer step) move 100 steps turn 90 degrees change pen color by 10 turn 5 degrees
Result: 100 rotating squares drawn one by one, producing a colourful spiral pattern.

Arrays (Lists in Scratch)

Section 3.7

Why Arrays?

If you need to store many values, using individual variables (marks_1, marks_2, … marks_50) becomes unmanageable. An array stores many items under a single name, using an index to refer to each item.

An array is a data structure that can hold any number of values using a single variable name.
❌ Without arrays (messy)
marks_1 = 78 marks_2 = 85 marks_3 = 69 marks_4 = 93 marks_5 = 87
✅ With arrays (clean)
Marks = [78, 85, 69, 93, 87] item 1 of Marks → 78 item 3 of Marks → 69

Building an Array in Scratch

  1. Go to the Data section in Scripts
  2. Click "Make a List"
  3. Give the array a name (e.g., Animal)
  4. Select "For this sprite only" or "For all sprites"
  5. Click OK
After creating a list, Scratch shows you blocks to add, delete, insert, replace and retrieve items.

Key Array Operations in Scratch

BlockWhat it does
add [thing] to [List]Appends an item at the end
delete (1) of [List]Removes item at position 1
insert [thing] at (1) of [List]Inserts at a specific position
item (1) of [List]Retrieves the item at position 1
length of [List]Returns how many items are in the list
[List] contains [thing]?Checks if a value exists in the list

Entering & Displaying Array Data

To add items (e.g., 5 country names):

delete all of Country repeat 5 ask "Enter the Country" and wait add answer to Country

To display all items:

set v_con to 0 repeat length of Country say (item v_con of Country) for 2 secs change v_con by 1
The variable v_con acts as the index — it steps through each position in the array.

3.8 – Problem Analysis with Arrays

Large problems are easier to solve when broken into smaller steps. Example: finding the average of 10 numbers.

  1. Input – ask for 10 numbers and store them in array Number
  2. Total – loop through the array and sum all values
  3. Average – divide total by 10
  4. Output – display the average
