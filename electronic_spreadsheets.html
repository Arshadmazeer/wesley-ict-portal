🔑 Key Terms

Workbook

A spreadsheet file containing one or more worksheets.

Worksheet

A single sheet made of rows and columns.

Cell

The box formed where a row and column meet.

Formula

An expression used to calculate values.

Function

A built-in formula such as SUM or AVERAGE.

Chart

A graphical way to display spreadsheet data.

What is a Spreadsheet?

A spreadsheet is an electronic document used to organize, store, and calculate data in a table made of rows and columns.

Data is entered into cells, and spreadsheets can perform calculations, sort data, and create graphs automatically.

Using Formulas

Formulas are used to calculate values in spreadsheets. Every formula starts with the = symbol.

Common Operators

Operation Symbol Example
Add+=C2+D2
Subtract-=E2-F2
Multiply*=I3*J3
Divide/=G3/H3

Examples

=C2+D2

Finds the total of Saturday and Sunday values.

=E2-F2

Finds the balance after deducting printing cost.

Using Functions

Functions are built-in formulas that help you calculate quickly over a range of cells.

SUM

Adds all numbers in a range.

=SUM(C3:G3)

AVERAGE

Finds the average value in a range.

=AVERAGE(C3:G3)

MAX

Shows the highest value in a range.

=MAX(C3:C12)

MIN

Shows the lowest value in a range.

=MIN(C3:C12)

COUNT

Counts cells that contain numbers only.

=COUNT(C3:C12)

COUNTA

Counts cells that are not empty.

=COUNTA(C3:C12)
